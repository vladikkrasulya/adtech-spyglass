Что отслеживается, а что нет

Статистика «Активность» в кабинете берётся из маленького per-user лога, который сохраняет ТОЛЬКО метаданные ниже. Тела bid request и bid response НИКОГДА не сохраняются в этот лог — они только проходят через валидатор, findings выходят, тела отбрасываются. (Сохранённые образцы библиотеки отдельно сквозно зашифрованы; см. Security.)

Сохраняется timestamp · oRTB версия · статус (clean/warnings/errors) · формат (banner/video/native/...) · количество findings (errors / warnings / total)
Нет в этом логе тела payload · ID отдельных findings · IP · user-agent · идентификатор партнёра · timestamps с точностью меньше дня в агрегатах
Операционный лог запросов отдельный лог фиксирует время · метод · путь · статус · твой IP (каждую ошибку плюс выборку успешных) — никогда payload
Анонимные вызовы нет per-user строки активности; операционный лог запросов и обезличенная аналитика всё равно применяются
Удаление аккаунта лог удаляется каскадно (foreign key)

Корпус поведения

Сохранённые event-streams с behavior-вкладки, помеченные как легитимные / мошеннические / неоднозначные. Confusion matrix ниже считает precision/recall каждого детектора на твоём же корпусе.

Confusion matrix · precision / recall

Каждый паттерн прогоняется через corpus как fraud-детектор: TP — сработал на «мошенничество», FP — сработал на «легитимно», FN — пропустил «мошенничество». Сорт по F1.

Настройки

Смешанное хранилище. Тема и диалект по умолчанию живут только в этом браузере (localStorage), без cross-device синхронизации. Язык findings по умолчанию также сохраняется в ваш аккаунт, когда вы залогинены (server-side, через /api/auth/preferences), поэтому переносится между устройствами.

Шифрование и восстановление

Библиотека зашифрована ключом, деривируемым из твоего пароля. Recovery key (32-символьный hex) показывали один раз при регистрации — храни его в password manager. Если потеряешь оба — библиотеку можно только стереть, не восстановить.

Действия с аккаунтом

Sign out очищает session cookie и блокирует библиотеку. Чтобы полностью удалить аккаунт, пиши на hi@ortbtools.com — автоматическое удаление сознательно не выставлено пока (редкое, необратимое, легко случайно кликнуть).
